# Screenshots

Please add the following screenshots to this folder:

1. **dashboard.png** - Main application dashboard showing all panels
2. **reports-list.png** - Reports list after fetching SLOOS data
3. **analysis-result.png** - Example of AI analysis (Summary, Trends, etc.)
4. **chat-interface.png** - Chat interface with AI assistant

## How to Take Screenshots

1. Open the application at http://localhost:7251
2. Click "Fetch SLOOS Data" and wait for reports to load
3. Take screenshot of the full page (dashboard.png)
4. Take screenshot showing the reports list (reports-list.png)
5. Select a report and click "Summary" or "Trends"
6. Take screenshot of the analysis result (analysis-result.png)
7. Ask a question in the chat and take screenshot (chat-interface.png)

## Recommended Tools

- **Windows**: Win + Shift + S (Snipping Tool)
- **Mac**: Cmd + Shift + 4
- **Browser**: F12 → Device Toolbar → Screenshot icon

Save all screenshots in this folder with the names above.
